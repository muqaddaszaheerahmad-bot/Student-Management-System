# Student Management System

## Overview

The Student Management System is a C++ application developed using Object-Oriented Programming (OOP) principles to efficiently manage and organize student records.

## Features

* Add new student records
* Update existing student information
* View student records
* Search for student details
* Manage student data efficiently

## Technologies Used

* C++
* Object-Oriented Programming (OOP)
